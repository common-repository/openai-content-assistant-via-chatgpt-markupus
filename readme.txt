ChatGPT - OpenAI helps to create a title & an excerpt based on the content of post using ChatGPT API.

== Description ==

The plugin extends the Gutenberg sidebar settings with two buttons that help you create titles and an excerpt for your content based on a blog post.

== Screenshots ==

1. Login to your OpenAi profile and generate a new key (https://platform.openai.com/account/api-keys).
2. Fill in the API token field with your key.
3. Select ChatGPT model if needed.
4. After saving the settings, you will receive a notification.
5. On the single page of the blog, enable the plugin and you will see the settings icon.
6. Click on the options and select "Create titles" or "Create excerpts".
7. Based on the content you will be offered the best options.
8. The last reply you received is stored in history. You can click on any option and it will be substituted in the title or excerpt of current post, respectively.
